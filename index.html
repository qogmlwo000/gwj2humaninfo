툴(Tool) 소개: 목적 및 주요 기능

🎯

목적

  • 단기 사원 공정 배정 자동화
  • 바코드 스캔을 통한 즉각적인 안내
  • 관리자 실시간 모니터링
⚙️

주요 기능

  • 센터별 엑셀 파일 업로드/통합
  • OB 여성 자동 층 배정 (백로그 기반)
  • 실시간 스캔 현황 대시보드
📝

핵심 관리

  • 현장 수동 변경 (즉시 적용)
  • 미스캔 인원 모트 확인
  • 데이터 백업 및 복원

워크플로우: 전체 사용 흐름

[관리자] 준비

설정 진입, 2/5센터 엑셀 업로드

(선택) 백로그 입력

[관리자] 배정

'인원 배정하기' 클릭

시스템 준비 완료 및 자동 백업

[사원] 안내

화면에서 바코드 스캔, 배정된 공정/층 확인

배정된 공정/층 확인

[관리자] 운영

실시간 현황 모니터링

(필요시) 수동 변경 및 엑셀 저장

STEP 1: 사전 준비

1. 엑셀 파일 업로드 (필수)

2센터 파일

'2센터' 이름이 포함된 파일

5센터 파일

'5센터' 이름이 포함된 파일

주의사항:
  • 반드시 '3. Simulation' 시트가 존재해야 합니다
  • '신규' 인원은 데이터 로드 시 자동으로 제외됩니다

2. (선택) 백로그 입력

2센터 OB 여성 백로그

MOS에서 층별(6.1+6.3, 7.2, 7.3) 백로그 수량을 확인하여 기입합니다.

────────────────────────

✅ MOS 설정 방법 추가 예정

────────────────────────

층별(6.1+6.3, 7.2, 7.3) 백로그 수량에 비례하여 자동 분배합니다

비워둘 경우: 층별 1/N 균등 분배됩니다

2센터 OB 여성 자동 배정 로직

CASE 1: 백로그 수량 입력 (권장)

입력된 백로그 *비율*에 맞춰 인원을 자동 분배합니다

[예시: 총 OB 여성 50명]
  • 6.1층 (2000) → 20% → 10명
  • 7.2층 (3000) → 30% → 15명
  • 7.3층 (5000) → 50% → 25명

CASE 2: 백로그 수량 미입력

3개 층에 *1/N으로 균등 분배*합니다

[예시: 총 OB 여성 50명]
  • 6.1층 → 17명
  • 7.2층 → 17명
  • 7.3층 → 16명

STEP 2: 현장 운영 (모니터링)

📊

실시간 현황판

전체 인원 대비 스캔 완료 인원 및 진행률(%)을 실시간으로 표시합니다. 사원 스캔 시 자동 업데이트됩니다.

👥

미스캔 인원 보기

아직 스캔하지 않은 인원들을 *공정별*로 그룹화하여 보여줍니다. 현장 미배치 인원 추적에 용이합니다.

✅

OB 배정 현황

2센터 OB/CFC OB 인원의 층별 배정 명단과 실시간 스캔(✓) 상태를 요약하여 보여줍니다.

STEP 2: 공정/층 수동 변경 (핵심 기능)

목적

📋
엑셀 데이터 오류 수정

업로드된 엑셀 파일의 오류를 즉시 수정할 수 있습니다

🔄
현장 상황에 따른 긴급 인원 재배정

실시간으로 발생하는 현장 상황에 즉각 대응합니다

⚠️
'원바코드 중복' 오류 사원 강제 배정

중복 바코드로 인한 문제를 현장에서 해결합니다

사용 화면

사용법

  1. 사원의 5자리 또는 8자리 바코드 입력
  2. 변경할 '공정' 또는 '층' 선택 (둘 다 선택 가능)
  3. '적용' 버튼 클릭 → *즉시 반영*
  4. '되돌리기'로 엑셀 원본 값 복구 가능

OB/CFC OB 엑셀 저장 기능

목적

2센터 OB/CFC OB 배정 결과를 현장 관리자 공유용 보고서로 생성합니다.

층별 시트 자동 분리

📑
층별 시트 자동 생성

'1.3층', '6.1층', '6.2층' 등 배정된 층별로 시트(Sheet)가 자동으로 분리되어 생성됩니다

바코드 누락 인원 식별

8자리 원바코드가 '누락'된 인원이 하단에 정렬됩니다.

바코드 중복 인원 식별

중요

'비고'란에 '원바코드 중복'이 표시됩니다. (관리자 확인 필요)

STEP 3: 사원 안내 (스캔 성공)

🎯

스캔 성공 (7초간 표시)

📍
표시 정보

센터 / 공정, 성함 (마스킹 처리, 예: 홍*동), 이동 층, 성공 사운드(딩동) 재생

스캔 오류 화면

오류: 명단 없음 (10초)

엑셀 명단에 *존재하지 않는* 바코드를 스캔한 경우입니다

안내사항
  • HUB 공정 가능성 안내
  • 1.1층 또는 4층 식당(HR)으로 이동 안내
  • 경고 사운드 재생

오류: 원바코드 중복 (10초)

동일한 5자리 바코드를 가진 인원이 *2명 이상*인 경우입니다

안내사항
  • 관리자에게 문의하도록 안내
  • (관리자 해결법: '수동 변경'으로 강제 배정)
  • 경고 사운드 재생

기타 기능: 백업 및 테스트

💾

데이터 백업 및 복원 (중요)

'인원 배정하기' 시 1차 *자동 백업*됩니다

'💾 백업 저장' (수동): 스캔 현황, 수동 변경 내역까지 모두 저장됩니다

새로고침(F5)해도 📁 백업 불러오기'로 즉시 복원 가능합니다

✏️

테스트 모드

엑셀 파일 없이 가상 데이터로 즉시 '인원 배정하기'를 실행합니다

발표, 시연, 또는 기능 테스트 시 사용합니다
